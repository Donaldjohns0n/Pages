# Meta-Organization Tracker

## Meta-Organization Reminder
**All agents and orchestration assets must honor dimensional awareness and maintain active cross-references to the central tracker.**

## Purpose
The meta-organization tracker provides a living, multidimensional map of every perspective, artifact, and process within the Pages framework. It unifies operational awareness, ensures consistent context retention, and supplies traceable audit trails for every agent interaction.

## Dimensional Awareness Protocol
1. **Pre-Interaction Alignment**: Agents must execute the "Pre-Interaction Dimensional Check" prompts before every engagement.
2. **Post-Interaction Reflection**: Agents log outcomes, anomalies, and cross-dimensional impacts using the "Post-Interaction Dimensional Check" prompts.
3. **Persistent Reminder**: Every blueprint and template begins with the Meta-Organization Reminder banner.
4. **Context Synchronization**: Updates to any dimension trigger references in the tracker table and are recorded in `/meta-log.md`.
5. **Audit Logging**: All dimensional shifts are annotated with timestamps, responsible agents, and affected artifacts.
